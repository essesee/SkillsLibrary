# Deadwood Phrase Table

A practical reference of common bloated phrases and their replacements. Apply during Pass 1 (Deadwood Elimination) and Pass 5 (Word Choice).

## Hedging & Qualifier Phrases

| Deadwood | Replace With | Example |
|----------|--------------|---------|
| "rather" | Delete | ~~rather~~ important → important |
| "very" | Delete or strengthen verb | very good → excellent |
| "quite" | Delete | ~~quite~~ similar → similar |
| "fairly" | Delete | ~~fairly~~ clear → clear |
| "somewhat" | Delete | ~~somewhat~~ better → better |
| "basically" | Delete | ~~basically~~ working → working |
| "actually" | Delete | ~~actually~~ happened → happened |
| "literally" | Delete | ~~literally~~ destroyed → destroyed |
| "simply" | Delete | ~~simply~~ put → (rephrase) |
| "truly" | Delete | ~~truly~~ great → great |
| "certainly" | Delete | ~~certainly~~ true → true |
| "definitely" | Delete | ~~definitely~~ will → will |
| "clearly" | Delete | ~~clearly~~ shows → shows |
| "I think" | Delete at sentence start | I think we should → We should |
| "in my opinion" | Delete | In my opinion, this works. → This works. |

## Wordy Constructions

| Deadwood | Replace With | Example |
|----------|--------------|---------|
| "due to the fact that" | "because" | Due to the fact that costs rose → Because costs rose |
| "in order to" | "to" | In order to improve → To improve |
| "with the exception of" | "except" | With the exception of John → Except John |
| "in spite of the fact that" | "although" | In spite of the fact that it's hard → Although it's hard |
| "for the purpose of" | "to" or "for" | For the purpose of testing → To test |
| "is able to" | "can" | Is able to work → Can work |
| "is in a position to" | "can" | Is in a position to decide → Can decide |
| "it goes without saying that" | Delete | It goes without saying that this matters. → This matters. |
| "there is no doubt that" | Delete | There is no doubt that it works. → It works. |
| "as a matter of fact" | Delete or "in fact" | As a matter of fact, true. → True. or: In fact, true. |
| "the fact that" | Often delete or rephrase | The fact that he left matters. → His departure matters. |

## Noun Phrase Bloat

| Deadwood | Replace With | Example |
|----------|--------------|---------|
| "at this point in time" | "now" | At this point in time → Now |
| "at the present time" | "now" | At the present time → Now |
| "in the final analysis" | "finally" | In the final analysis → Finally |
| "prior to" | "before" | Prior to launch → Before launch |
| "subsequent to" | "after" | Subsequent to the meeting → After the meeting |
| "for a period of" | Delete | For a period of three weeks → For three weeks |
| "of great importance" | "important" | Is of great importance → Is important |
| "of a nature that is" | Delete or rephrase | Is of a nature that is complex. → Is complex. |
| "to be in agreement" | "to agree" | To be in agreement → To agree |
| "give consideration to" | "consider" | Give consideration to → Consider |

## Passive Voice Heavy Constructions

| Deadwood | Replace With | Example |
|----------|--------------|---------|
| "is able to be" | Active equivalent | Is able to be fixed → Can be fixed (or: Can fix) |
| "there is" (opening) | Active subject | There is a bug in the code. → A bug exists in the code. or: The code has a bug. |
| "it is" (opening) | Active subject | It is important to note that... → Note that... |
| "is found to be" | "is" or restate | Is found to be useful → Proves useful |
| "could be argued that" | Direct claim | Could be argued that it's true. → It's true. |
| "was determined to be" | Active verb | Was determined to be effective. → Proved effective. |

## Redundant Pairs & Clichés

| Deadwood | Replace With | Example |
|----------|--------------|---------|
| "first and foremost" | "first" | First and foremost → First |
| "one and only" | Delete | The one and only solution → The solution |
| "each and every" | "each" or "every" | Each and every person → Each person |
| "null and void" | "null" or "void" | Null and void → Void |
| "end result" | "result" | The end result is → The result is |
| "final outcome" | "outcome" | The final outcome shows → The outcome shows |
| "true facts" | "facts" | True facts about → Facts about |
| "close proximity" | "near" | In close proximity to → Near |
| "large in size" | "large" | Large in size → Large |
| "round in shape" | "round" | Round in shape → Round |
| "in the final analysis" | Delete or "finally" | In the final analysis, true. → True. |

## Vague Intensifiers

| Deadwood | Replace With | Example |
|----------|--------------|---------|
| "really" | Delete or strengthen | Really good → excellent |
| "kind of" | Delete or specify | Kind of broken → broken (or: partially broken) |
| "sort of" | Delete or specify | Sort of worked → worked (or: partially worked) |
| "a bit" | Delete or specify | A bit slow → slow (or: 20% slower) |
| "pretty" | Delete | Pretty important → important |
| "somewhat" | Delete or specify | Somewhat larger → larger (or: 15% larger) |

## Usage Examples

### Before (Deadwood)
"Due to the fact that the system is able to be improved in order to provide a somewhat better user experience, we should, quite frankly, give consideration to this proposal, which is, in the final analysis, of great importance for the organization."

### After (Edited)
"Because we can improve the system's user experience, we should consider this proposal. It matters to the organization."

**Changes:**
- "Due to the fact that" → "Because"
- "is able to be improved" → "improve"
- "in order to" → (deleted, restructured)
- "somewhat" → (deleted)
- "quite frankly" → (deleted)
- "give consideration to" → "consider"
- "which is, in the final analysis, of great importance" → "It matters"
- Sentence count reduced from 1 to 2, clarity improved
