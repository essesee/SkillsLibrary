# WCAG 2.1 AAA Gap Analysis — Beyond AA

AAA is aspirational. These are improvements beyond the AA baseline. Label findings as "AAA improvement opportunity," not failures.

## Enhanced Contrast (1.4.6)
- Text contrast 7:1 for normal text, 4.5:1 for large text
- Higher than AA minimums — audit current contrast ratios and flag those between AA and AAA thresholds

## Extended Audio Description (1.2.7)
- Audio description provided for all prerecorded video content (not just when pauses allow)

## Sign Language (1.2.6)
- Sign language interpretation for prerecorded audio

## No Timing (2.2.3)
- Timing is not an essential part of any interaction (stricter than AA's "can be extended")

## No Interruptions (2.2.4)
- Interruptions can be postponed or suppressed entirely (notifications, alerts)

## Re-authentication (2.2.5)
- If session expires during data entry, data is preserved after re-authentication

## Three Flashes (2.3.2)
- No content flashes at all (stricter than AA's 3-per-second limit)

## Navigation Enhancements
- **Location (2.4.8):** User's location within the site is always indicated (breadcrumbs)
- **Link Purpose (2.4.9):** Link purpose determinable from link text alone (no surrounding context needed)
- **Section Headings (2.4.10):** Content organized with headings at appropriate levels

## Reading Level (3.1.5)
- Content readable at lower secondary education level (roughly grade 7-9)
- Supplemental content or simplified version provided for complex text

## Pronunciation (3.1.6)
- Mechanism available for determining pronunciation of ambiguous words

## Change on Request (3.2.5)
- Context changes only on user request — no automatic redirects, no auto-submits

## Help (3.3.5)
- Context-sensitive help available on every page

## Error Prevention — All (3.3.6)
- All submissions (not just financial/legal) are reversible, checked, or confirmed
