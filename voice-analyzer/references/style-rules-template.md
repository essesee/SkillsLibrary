# Elements of Style Overlay — Template

This file layers on top of the voice profile. Where the user's natural voice conflicts with Strunk & White, note the tension and sharpen rather than replace.

## Rules Applied During Drafting

1. **Omit needless words.** Trim filler without losing the user's rhythm.
2. **Active voice by default.** Convert passive unless the user habitually uses passive for specific purposes (diplomacy, blame-diffusion).
3. **Positive form.** "We should X" not "We shouldn't not X."
4. **Specific over vague.** Numbers, names, dates where possible.
5. **Parallel structure.** Lists and paired expressions match in form.

## User-Specific Tensions

Document tensions here after voice profile analysis:

- [TENSION]: User tends to [habit] but Strunk & White recommends [alternative]. Resolution: [sharpen/preserve/hybrid].

Example:
- TENSION: User opens emails with context-setting paragraphs. Strunk prefers BLUF. Resolution: Preserve context-first for exec audiences (it's their style), tighten to BLUF for peers.

## Toggle

- **ON (default):** Apply style rules during drafting. Voice profile + style overlay.
- **OFF:** Voice profile only. No style filtering. Use for quick casual Slack messages where tightness doesn't matter.
